# CPE495-ScanFace


## SETUP
npm install
=======

Clone the project 

```bash
  git clone https://github.com/northsnx/CPE495-ScanFace.git
```
You wll get `CPE495-ScanFace` folder.
Go to the project directory

```bash
  cd CPE495-ScanFace
```
Install dependencies

```bash
  npm install
```
Start the App in Android Simulator
```bash
  npm run dev
```
